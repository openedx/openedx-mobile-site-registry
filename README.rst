Experimental Mobile Site Registry
#################################

This repository will hold an **experimental** implementation of a site registry backend for the
`proposed Universal Open edX Mobile Apps <https://openedx.atlassian.net/wiki/spaces/OEPM/pages/6489505793/DRAFT+-+Joint+Governance+Agreement+Universal+Mobile+Learning+App+Initiative>`_.

**The Idea**
  One or more community providers would host the registry.
  Open edX instances could apply to be listed on the registry.
  The Universal Mobile App would allow users to enroll in courses across all sites on the registry.

**Open Questions**
  * Will there be one registry or multiple?
  * Who will host the registry/registries?
  * How will it be decided which instances are accepted onto the registry/registries?
  * Will the Universal app be Open edX branded or not?

**Status**
  This is `still undergoing the product review process <https://github.com/openedx/platform-roadmap/issues/516>`_.
  We have create the repo so contributors have a centralized place to experiment.
  If the Universal Mobile App proposal is not accepted, we will archive this repo.
